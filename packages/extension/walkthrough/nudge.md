# Optional: get nudged

If you want the IDE to remind you, set **ClankyBuddy → Nudge: Interval Minutes** in Settings (try 60 or 120). A small toast pops in the bottom-right with `Open`, `Snooze 1h`, and `Turn off`.

Leave it at `0` for no nudges at all. The status bar entry and command palette are always there when you want them.
